# TaskMaster

![Demo截图](截图URL) <!-- 可选 -->

## 功能特性
- ✅ 用户注册/登录（密码加密）
- ✅ 任务CRUD操作
- ✅ 状态流转（待办/进行中/完成）
- ✅ 响应式布局
- ✅ 实时交互反馈

## 技术栈
| 模块       | 技术                 |
|------------|----------------------|
| 前端       | Bootstrap 5 + JS     |
| 后端       | Python Flask         |
| 数据库     | SQLite               |
| 部署       | 支持Docker（可选）   |

## 快速启动
```bash
pip install -r requirements.txt
flask run
